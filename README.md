# Hand - drawn Area UAV Full - coverage Path Planning System

## Project Introduction

This is a system that can automatically convert arbitrarily shaped hand - drawn areas into full - coverage flight paths for unmanned aerial vehicles (UAVs).

## Functional Features

- Supports the recognition of areas in any hand - drawn shape.
- Automatically extracts the contour and generates an optimal coverage path.
- Supports various scanning directions and start - point optimization.
- Automatically calculates the total flight distance and the distance of each section.

## Usage

1. Save the hand - drawn area image as `hand_drawn.png` in the root directory of the project.
2. Run `main.py`.
3. The generated path map will be saved as `drone_path_result.png`.
